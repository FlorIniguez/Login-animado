# Proyecto: Animación de Monstruo Interactivo con Contraseña (en Login y Register)

Este proyecto es una aplicación web interactiva creada con **HTML**, **CSS**, y **JavaScript**. La aplicación presenta un personaje de monstruo animado que reacciona de manera dinámica a las acciones del usuario, como la introducción de una contraseña, el seguimiento del puntero del mouse y la visualización de la contraseña.

## Características

### Interacción con el input de contraseña:
- El monstruo cubre sus ojos cuando el usuario comienza a escribir la contraseña.
- Al mostrar la contraseña, el monstruo se destapa los ojos.
- Las imágenes del monstruo cambian dependiendo del estado de la contraseña (visible u oculta).

### Seguimiento del puntero del mouse:
- El monstruo sigue el movimiento del mouse en la pantalla, con diferentes imágenes dependiendo de la posición del puntero.

### Animación basada en la longitud del texto:
- Mientras se escribe en los inputs de correo electrónico o nombre de usuario, el monstruo cambia sus expresiones dependiendo de la longitud del texto ingresado.
